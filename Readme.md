
 <h3 align="center">Cắt PSD sang HTML</h3>
Yêu cầu:
* Code giao diện hiển thị tốt trên Chrome, Firefox<br>
* Giao diện hiển thị Responsive<br>

* Đặt tên trong HTML CSS chuẩn BEM<br>
* Sử dụng Fontawesome để dùng cho Icons<br>
